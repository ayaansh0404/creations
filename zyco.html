Install Zyco

Put this code in your HTML head to use Zyco:

<script>
fetch("https://gist.githubusercontent.com/ayaansh0404/575a9ebfef498b2bc557307b311a02c2/raw/zyco.css")
.then(res => res.text())
.then(css => {
const style = document.createElement("style");
style.textContent = css;
document.head.appendChild(style);
});
</script>
